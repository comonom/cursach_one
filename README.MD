# Курсовой проект

## Запуск проекта
Для того, чтобы работать с приложением, необходимо указать данные для подключения к БД.
При разработке мы пользуемся UserSecrets, отредактировать их можно через контекстное меню проекта - "Управление секретами пользователей". Формат указан ниже. Данные из этого файла не попадают в /bin или git, они расположены локально на компьютере.
Для использования бинарников необходимо изменить appsettings.json.  В разделе "ConnectionString" создается поле "db" со значением, содержащим строку подключения.
Пример конфигурации в appsettings.json (или secrets.json при работе с UserSecrets):
``` json
{
	"ConnectionString": {
		"db": "server=localhost;user id=admin;password=admin123;database=simbirsoft"
	}
}
```
